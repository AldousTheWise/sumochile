# Sumo Chile – Landing Page

Landing page oficial para **Sumo Chile**, desarrollada con **Vue 3** y **Vite**, enfocada en una estética brutalista, alto impacto visual y una experiencia clara tanto en desktop como en mobile.

El proyecto está pensado como base escalable para futuras funcionalidades como autenticación, panel de administración y gestión de contenidos.

---

## Tecnologías utilizadas

- Vue 3 (Composition API)
- Vite
- Vue Router
- Pinia
- Tailwind CSS
- HTML5 / CSS3
- JavaScript (ES6+)

---

## Características principales

- Landing page responsive (desktop y mobile)
- Carrusel de imágenes como fondo
- Diseño brutalista con elementos geométricos y diagonales
- Uso de tipografías personalizadas:
  - Anton (títulos)
  - Outfit (textos)
- Arquitectura preparada para crecimiento futuro
- Estado global gestionado con Pinia

## Instalación y ejecución

### Requisitos
- Node.js v18 o superior

### Pasos

1. Clonar el repositorio:
   ```bash
   git clone https://github.com/AldousTheWise/sumochile.git
   ```
   
Entrar al proyecto:
```bash
cd sumochile
```

Instalar dependencias:
```bash
npm install
```

Ejecutar en desarrollo:
```bash
npm run dev
```

Compilar para producción:
```bash
npm run build
```
